### Commands and steps
- Clone the repository: `git clone ...`
- Change directory: `cd roominghouseapp_be`
- Install dependencies: `npm install`
- Create a .env file and fill
- Run migration: `npm run migration:run`
- Start the project: `npm run start:dev`
